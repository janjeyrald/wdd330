    Reading Notes

    It doesn't mean that your website is responsive it doesn't mean it's mobile first.
    We need to design a mobile first website.
    Eric Schmidt from Google said " The simple guideline is whatever you are doing, do mobile first".
    Responsive retrofitting from desktop to mobile devices.

      Speed Matters
    • Reduce Image usage throughout sites/apps
    • SVG/webfonts are lower bandwidth
    • optimize and mininfy CSS and JS files
    • GZIP files to eliminate unnecessary data
      Mobile advertising user experience
    • Stop using Pop-ups/pop-overs
    • User don't visit your website to look at ads
    • Keep it simple and sleek
    • Make sure it's unobtrusive.

    Questions:

    1. What is the ideal size for small, medium and large screens?
    2. What unit should I use on my media query? Is it px, em or rem?
